# TextRainCanvas(oCanvas, option)
黑客帝国特效，网上的代码不可调节背景色和速度，自己实现了一个



#### @参数

| 参数        | 描述  | 
| --------   | -----:  |
| oCanvas      | canvas容器   |
| option      | @配置项  |


#### @配置项

| 参数        | 描述  | 
| --------   | -----:  |
| text      | 需要展示的文本（随机展示）   |
| fontSize      | 字体大小，默认 14  |
| color      | 颜色。例：#00ff00  |
| reverse      | 是否倒放。true为从下到上，false为从上到下  |
| interval      | 绘制速度，默认 100  |

#### @方法

| 参数        | 描述  | 
| --------   | -----:  |
| dispose      | 销毁   |
| start      | 开始播放  |
